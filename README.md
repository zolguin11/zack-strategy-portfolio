# Zack Olguin Strategy Portfolio

Static portfolio site for Zack Olguin.

## Local Preview

Run:

```bash
node server.js
```

Then open:

```text
http://127.0.0.1:3000
```

## Vercel

This is a static HTML/CSS/JS site. In Vercel, use:

- Framework preset: Other
- Build command: leave blank
- Output directory: `.`
